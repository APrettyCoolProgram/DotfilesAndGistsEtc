# About appdata/
Codebase assets, documentation, and data.

**Created at runtime:** Yes, always.

| **/**        |                                                                                                      |
|:------------ |:---------------------------------------------------------------------------------------------------- |
| **cache/**   | Cached data                                                                                          |
| **conf/**    | Configurations and settings                                                                          |
| **dev/**     | Development resources                                                                                |
| **doc/**     | Documentation                                                                                        |
| **export/**  | Exported data                                                                                        |
| **font/**    | Fonts                                                                                                |
| **image/**   | Images                                                                                               |
| **import/**  | Imported data                                                                                        |
| **runtime/** | Runtime data                                                                                         |
| **temp/**    | Temporary data                                                                                       |
